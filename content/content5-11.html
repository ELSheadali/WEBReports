Висновки:

В даній роботі було розглянуто функціональне застосування JavaScript у HTML-документі. Досліджено використання масивів у JavaScript-сценаріях та реалізацію програм засобами мови JavaScript.

Використання масивів у JavaScript-сценаріях дозволяє зручно та ефективно обробляти та управляти колекціями даних. Масиви в JavaScript можуть містити різні типи даних та об'єкти, що розширює їхню функціональність. Завдяки широкому набору вбудованих методів масивів, таких як push, pop, splice та інші, розробники можуть легко виконувати різноманітні операції з даними.

Реалізація програм засобами мови JavaScript відкриває безліч можливостей для створення інтерактивних веб-додатків, ігор, веб-сайтів та інших веб-застосунків. JavaScript дозволяє створювати динамічний контент, обробляти події користувача, взаємодіяти з сервером та іншими скриптами на стороні клієнта.

Усі три завдання були успішно виконані, що показує вміння використовувати JavaScript для створення функціональності на веб-сайті та для реалізації корисних інструментів для користувача.
